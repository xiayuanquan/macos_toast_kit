# macos_toast_kit

A new Flutter plugin project.

## Getting Started

Flutter Plugin, Support System Toast similar to macOS

* 1.support application mode
  * 0 : light mode
  * 1 : dark mode
  * 2 : follow the system

* 2.support position
  * 0 : show at center 
  * 1 : show at left
  * 2 : show at top
  * 3 : show at right
  * 4 : show at bottom
